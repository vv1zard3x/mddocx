Правила для LLM

Markdown contract

Структура markdown, при которой docxmd даёт детерминированный .docx. Загружай как rule в Cursor (.cursor/rules/) или в Claude (Project Knowledge / Custom Instructions / CLAUDE.md).

Cursor (.mdc) Claude (.md)

Prose style

Стилевые правила для длинных текстов: пунктуация, словарь по умолчанию, ритм предложений, конструкции, от которых стоит уходить.

Cursor (.mdc) Claude (.md)
Контракт оптимизирован для копирования в system-промпт LLM. Любой генератор, который ему следует, даёт детерминированный .docx через /convert/md или MCP-инструмент convert_md.
